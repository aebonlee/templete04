# Nova — 제품 랜딩페이지 디자인 템플릿

모던 SaaS/제품 소개용 **반응형 랜딩페이지 템플릿**입니다.
Raycast 스타일의 다크 테마와 그라디언트 강조 색상을 참고하여, 순수 HTML/CSS/JS로
프레임워크 없이 바로 사용할 수 있도록 제작했습니다.

## 특징

- 🎨 **다크 테마 + 그라디언트** — CSS 변수(디자인 토큰)로 색상·간격·반경을 한 곳에서 관리
- 📱 **완전 반응형** — 960 / 760 / 460px 브레이크포인트, 모바일 햄버거 메뉴
- 🧩 **재사용 컴포넌트** — 버튼, 카드, 그리드, 요금제, 후기, CTA 등
- ⚡ **의존성 없음** — 빌드 도구·외부 라이브러리 불필요, 파일만 열면 동작
- ♿ **접근성 고려** — 시맨틱 마크업, `aria` 속성, `prefers-reduced-motion` 지원

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 페이지 구조 (내비 · 히어로 · 기능 · 연동 · 후기 · 요금제 · CTA · 푸터) |
| `styles.css` | 디자인 토큰, 컴포넌트 스타일, 반응형 규칙 |
| `script.js`  | 모바일 메뉴 토글, 스크롤 진입 애니메이션 |

## 섹션

1. **Navigation** — 스티키 헤더 + 모바일 메뉴
2. **Hero** — 그라디언트 헤드라인 + 제품 미리보기 창(UI 목업)
3. **Logos** — 신뢰 로고 스트립
4. **Features** — 3열 기능 카드 그리드
5. **Integrations** — 4열 연동 카드
6. **Testimonials** — 고객 후기
7. **Pricing** — 3단 요금제 (추천 플랜 강조)
8. **CTA** — 뉴스레터 구독 폼
9. **Footer** — 링크 그룹 + 소셜

## 사용 방법

```bash
# 로컬에서 바로 열기
open index.html          # macOS
# 또는 간단한 서버로
python3 -m http.server 8000
```

## 커스터마이징

`styles.css` 상단의 `:root` 토큰만 바꿔도 전체 테마가 변경됩니다.

```css
:root {
  --brand:   #6c8cff;   /* 주요 강조색 */
  --brand-2: #a06bff;   /* 보조 그라디언트 색 */
  --accent:  #35d0a5;   /* 포인트 색 */
  --bg:      #0b0d12;   /* 배경 */
  --radius:  16px;      /* 카드 모서리 */
}
```
